# CDSS 3.0 启动指南

## 快速启动（开发环境）

### 前提条件
- Docker & Docker Compose 已安装
- Node.js 18+ 已安装
- Python 3.11+ 已安装（可选，Docker 内已包含）

---

### 第一步：配置环境变量
```bash
cp .env.example .env
# 按需修改 .env 中的配置（默认值可直接使用）
```

### 第二步：启动基础设施（数据库/ES/Redis）
```bash
# 仅启动依赖服务（不包含后端，方便本地调试）
docker-compose up -d postgres elasticsearch redis minio
```

### 第三步：启动后端
```bash
cd backend
pip install -r requirements.txt

# 初始化数据库并写入 Seed 数据
python -m mock.seed_data

# 启动 FastAPI 服务
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

后端启动后访问：
- API 文档：http://localhost:8000/api/docs
- 健康检查：http://localhost:8000/api/health

---

### 第四步：启动前端（三端分别启动）

**医生PC端（端口 3000）**
```bash
cd frontend/pc
npm install
npm run dev
# 访问：http://localhost:3000/pc/
```

**HIS嵌入端（端口 3001）**
```bash
cd frontend/his
npm install
npm run dev
# 访问：http://localhost:3001/his/
```

**管理端（端口 3002）**
```bash
cd frontend/admin
npm install
npm run dev
# 访问：http://localhost:3002/admin/
# 默认账号：admin / admin123
```

---

### 完整 Docker 一键启动（含前端构建后）
```bash
# 先构建三端前端
cd frontend/pc && npm install && npm run build
cd ../his && npm install && npm run build
cd ../admin && npm install && npm run build
cd ../..

# 启动全部服务
docker-compose up -d

# 初始化数据
docker-compose exec backend python -m mock.seed_data
```

访问地址：
- PC端：http://localhost/pc/
- HIS端：http://localhost/his/
- 管理端：http://localhost/admin/

---

## 外部系统 Mock 切换

`.env` 文件中切换（无需改代码）：
```env
HIS_MODE=mock   # → real 时启用真实接口
LIS_MODE=mock   # → real 时启用真实接口
```

## 目录结构速查

```
CDSS-3.0/
├── backend/
│   ├── main.py              # FastAPI 入口
│   ├── config.py            # 配置读取
│   ├── api/v1/              # 路由（按功能分文件）
│   ├── services/            # 业务逻辑
│   ├── services/adapters/   # 外部系统适配器（Mock/真实）
│   ├── db/models/           # SQLAlchemy 数据模型
│   ├── db/repositories/     # 数据访问层
│   ├── db/elasticsearch/    # ES 索引定义
│   ├── schemas/             # Pydantic 请求/响应模型
│   ├── mock/seed_data.py    # Seed 数据（初始化用）
│   └── tasks/               # Celery 异步任务
├── frontend/
│   ├── shared/              # 三端共享 API 封装
│   ├── pc/                  # 医生PC端（Vue3）
│   ├── his/                 # HIS嵌入端（Vue3）
│   └── admin/               # 管理端（Vue3）
├── docs/                    # 开发文档
├── docker-compose.yml
├── nginx.conf
└── .env.example
```
